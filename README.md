# 🛒 E-Commerce Sales Analysis & Customer Segmentation Dashboard

## 📊 Project Overview

Comprehensive analysis of 10,000 e-commerce transactions across 5 product categories, featuring advanced RFM (Recency, Frequency, Monetary) customer segmentation and an interactive Tableau dashboard for business intelligence.

---

## 🎯 Key Insights

- **$1,286,772** in total annual revenue analyzed (2023)
- **3,564 unique customers** segmented into 6 actionable behavioral groups
- **Sports category** dominates with 33% market share ($432K revenue)
- **24% of customers** classified as "Lost" - prime targets for re-engagement campaigns
- **23% Champions** identified as high-value customers driving sustainable revenue
- **Monthly sales variance** of 14% reveals seasonal purchasing patterns

## 📈 Analysis Workflow

### 1. Data Generation & Preparation
- Synthetic dataset creation with realistic e-commerce patterns
- 10,000 transactions across 5 categories
- 3,564 unique customers
- Date range: January - December 2023

### 2. Database Design
- SQLite database with normalized schema
- Efficient querying with indexed tables
- 6+ analytical SQL queries for insights extraction

### 3. RFM Customer Segmentation
Advanced behavioral segmentation based on:
- **Recency:** Days since last purchase
- **Frequency:** Total number of purchases
- **Monetary:** Total spend amount

### 4. Tableau Dashboard Development
- Interactive visualizations with cross-filtering
- Revenue analysis by category
- Customer segment distribution
- Monthly sales trend analysis
